Web Interface for the Synthesis package

## Installation

With `pip`:
```bash
pip install synthesis-web
```

With [`pipx`](https://github.com/pipxproject/pipx):
```bash
python3.7 -m pip install --user pipx
pipx install synthesis-web
```
